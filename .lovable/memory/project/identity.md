TrustLayer is an AI-powered on-chain identity and reputation layer.

It analyzes wallet activity (DeFi, NFT, DAO, social) to produce a 0-1000 credit score,
generates zero-knowledge proofs for selective disclosure, and lets users monetize
permissioned data access by dApps.

Core sections: Overview, Credit Score, Proofs, Earnings, Persona, Developers, Settings.

Mock-only frontend (no real Web3, no backend). Sui-blue primary palette. Mobile responsive.
